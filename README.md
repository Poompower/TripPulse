##  Core Idea
TripGuider คือแอปสำหรับ **วางแผนทริปท่องเที่ยวแบบครบในที่เดียว**  
ช่วยให้ผู้ใช้สามารถสร้างทริป กำหนดการเดินทางรายวัน ค้นหาสถานที่จาก External API และดูข้อมูลประกอบการเดินทางได้อย่างเป็นระบบ ลดความยุ่งยากจากการใช้หลายแอปพร้อมกัน

##  MVP Features

### 1) Trip Management
- สร้าง / แก้ไข / ลบทริป
- กำหนดชื่อทริป เมือง/ประเทศ วันเริ่ม–วันจบ
- แสดงรายการทริปทั้งหมด (Trip List)

---

### 2) Auto Day Itinerary
- ระบบสร้างวันของทริปอัตโนมัติตามช่วงวันที่
- แยก itinerary เป็นรายวันอย่างชัดเจน

---

### 3) Places Search (External API)
- ค้นหาสถานที่ท่องเที่ยวและจุดสนใจจาก **Geoapify Places API**
- กรองสถานที่ตามหมวดหมู่ เช่น tourist attraction
- แสดงข้อมูลพื้นฐานของสถานที่ เช่น ชื่อและตำแหน่ง

---

### 4) Add Place to Itinerary
- เพิ่มสถานที่ที่ค้นพบเข้าไปในวันของ itinerary
- ผูกสถานที่กับทริปและวันที่ที่เลือก

---

### 5) Itinerary Management
- แสดงรายการสถานที่ในแต่ละวัน
- จัดลำดับ (reorder) สถานที่ในวันนั้น
- ลบสถานที่ออกจาก itinerary ได้

---

### 6) Route Summary (External API)
- คำนวณระยะทางและเวลาเดินทางโดยประมาณระหว่างสถานที่
- แสดงผลในรูปแบบสรุป เช่น distance และ duration

---
